## كيفية التشغيل

1. **تأكد من وجود الملفات**:
   ```bash
   ls public/questions.all.json
   ls public/media/questions/*/default.png
   ```

2. **تشغيل اللعبة**:
   ```bash
   npm run dev
   ```

3. **الوصول للعبة**:
   افتح المتصفح على `http://localhost:5173`
